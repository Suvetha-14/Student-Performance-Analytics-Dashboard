# 📊 Student Performance Analytics Dashboard

## 📌 Project Overview
This project focuses on analyzing student data to identify performance trends and highlight students who may be at risk of failing or dropping out.

---

## 🎯 Objective
To analyze student data and provide insights that help institutions take early academic interventions.

---

## 📂 Dataset
The dataset includes:

- **Marks**
- **Attendance**
- **Login Activity**

---

## ⚙️ Analysis Performed

- Calculated **average performance metrics**
- Analyzed **correlation between attendance and marks**
- Studied the **impact of absenteeism**
- Identified **top-performing and struggling students**

---

## 📊 Power BI Dashboard

The dashboard includes:

- Bar charts for performance comparison  
- Heatmaps for trend analysis  
- Insights on student risk levels  
- Interactive visuals for better understanding  

---

## 📈 Outcome

- Identified **at-risk students early**
- Provided insights to support **academic decision-making**
- Helped improve **student performance monitoring**

---
